# CronSong
-------------

Pick and queue music for your weekly and daily standups. To be run on Raspberry PI, but values can be swapped to suit any linux environment

```
npm install
sudo apt-get install mplayer
gulp

```

in a new term
```
npm run start
```

build prod

```
gulp dist
```
